Flavours

Buttons come in 3 flavours, button-primary, button-secondary, and button-white. For inverted versions add --inverted to the end of the class name, e.g. button-primary--inverted.

Sizes

Four buttons sizes are available by adding an extra size modifier, button--l, default (no class required), button--s and button--xs. Note: You can use a size modifier with any of the button styles above.

Minimum or Maximum Width

For visual consistency you can also set a minimum or maximum width on buttons using button--set-min-width or button--set-max-width.
