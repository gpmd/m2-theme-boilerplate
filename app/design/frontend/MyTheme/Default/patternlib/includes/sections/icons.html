MyTheme Iconset

Example usage:

<div class="i-burger--s-black" data-grunticon-embed></div>

Colour Variants

Icons are available in four colours, black (actually dark grey in this case), grey, white and patina (green). Colours are denoted by adding a keyword to the end of the CSS class name, e.g. -black:

<div class="i-burger--s-black" data-grunticon-embed></div>

Size Variants

Some icons are available in two sizes, large and small. Sizes are denoted by adding a size modifier after the icon identifier in the CSS class name, e.g. --l:

<div class="i-burger--l-black" data-grunticon-embed></div>

States

Icons can have a hover state by adding the class i--has-hover (currently sets the opacity to 50%).

i-burger--s-black i--has-hover

i-search--s-black i--has-hover

i-user--s-black i--has-hover

i-bag--s-black i--has-hover
